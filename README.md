# SebbyScreensaver

A screensaver for Windows made with C#, .NET, and WinForms. <br>
It shows my logo which bounces around the screen. <br>
You can configure the size of the image with screensaver settings.

#### Notice: This screensaver is only compatible with Windows (8 and above).

## Installation instructions
1. Visit the [releases page](https://github.com/sebminecrafter/SebbyScreensaver/releases).
2. Download the file labeled "SebbyScreensaver.scr".
3. You can either:
   - Move the downloaded file to `C:\Windows\System32` and then select it through `Change Screen Saver` in Control Panel.
   - Or, right-click on the downloaded file and select `Install` to directly install it.
